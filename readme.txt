== Description ==

توسط آینکس می توانید امکانات خرید شارژ، پرداخت قبض، خرید بسته اینترنت را به سایت وردپرسی خود اضافه کرده و کسب درآمد کنید

= ویژگی ها : =

*	خرید شارز مستقیم برای اپراتورهای ایرانسل، همراه اول و رایتل
*	خرید شارژ پین
*	خرید بسته های اینترنتی
*	پرداخت قبوض آب، برق، گاز، تلفن همراه و تلفن ثابت، جریمه رانندگی
*	کسب درآمد از فروش کالا
*	بدون نیاز به شارژ حساب کاربری
*	اتصال مستقیم به درگاه پرداخت بانکی
